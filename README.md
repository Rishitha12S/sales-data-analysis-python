# Sales Data Analysis Project

## 📊 Project Overview

This project focuses on analyzing a retail sales dataset to uncover useful business insights such as sales performance, top-selling products, customer purchasing behavior, and monthly sales trends.

The analysis was performed using Python with data manipulation and visualization libraries.

---

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib

---

## 📂 Dataset Information

The dataset used in this project is the **Superstore Sales Dataset**, which contains:

* 9800 rows (sales records)
* 18 columns (order details, customer data, product information, and sales values)

Key fields include:

* Order Date
* Customer Name
* Region
* Category
* Product Name
* Sales

---

## 🔎 Analysis Performed

The following analysis was carried out:

1. **Total Sales Calculation**
   Calculated overall sales generated in the dataset.

2. **Region-wise Sales Analysis**
   Identified which regions generate the highest revenue.

3. **Category-wise Sales Analysis**
   Compared sales across product categories such as:

   * Furniture
   * Office Supplies
   * Technology

4. **Top 10 Products**
   Determined the highest selling products based on total sales.

5. **Top 10 Customers**
   Identified customers who contributed the most revenue.

6. **Monthly Sales Trend**
   Analyzed how sales change across different months.

---

## 📈 Visualizations

The project includes several data visualizations created using Matplotlib:

* Region-wise Sales Bar Chart
* Category Sales Bar Chart
* Monthly Sales Trend Line Chart

These visualizations help identify important patterns and trends in the dataset.
